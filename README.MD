# Teste Front-end Lime

Projeto front-end estático de e-commerce, com páginas de home, autenticação, categoria, produto e checkout.

## Deploy

Aplicação em produção:

- https://teste-front-end-sooty.vercel.app/

## Tecnologias utilizadas

- HTML5
- Tailwind CSS (CDN)
- JavaScript (Web Components)
- Swiper.js (carrosséis)
- Lucide Icons
- jQuery

## Como rodar o projeto localmente


### VS Code + Live Server

1. Abra a pasta do projeto no VS Code.
2. Instale a extensão Live Server,
3. Clique com o botão direito em `index.html` e selecione **Open with Live Server**.
4. O navegador abrirá automaticamente em uma URL local.

Depois abra:

- http://localhost:5500/index.html

## Páginas disponíveis

- `index.html` - Home
- `login.html` - Login
- `cadastro.html` - Cadastro
- `categoria.html` - Categoria
- `produto.html` - Detalhe do produto
- `checkout.html` - Checkout
